USO DE JUEGOS Y EL ALFABETO:
(UNICAMENTE PARA SISTEMA OPERATIVO WINDOWS)
Para poder usar los juegos y el alfabeto, es necesario instalar el Navegador BAIDU.
En el paquete se provee el programa instalador del Navegador BAIDU.
PARA INSTALAR: DAR CLICK AL BOTÓN DE ARRIBA "INSTALAR NAVEGADOR BAIDU".
Es necesario buscar el archivo :
baidu-spark-browser.exe dar Doble-click para instalar.
Establecerlo como predeterminado.
Luego, es necesario buscar el archivo ".exe" en la carpeta correspondiente,
ya sea ALFABETO O JUEGOS, y dar Doble-click a ese archivo ".exe" para poder abrir el programa de Alfabeto o Juegos desde ahi.

PARA TODAS LAS DEMAS FUNCIONES, AUDIO, VIDEO Y SPELLING BEE SE RECOMIENDA GOOGLE CHROME.

PARA USAR LA PLATAFORMA CON GOOGLE CHROME:
1. ESTABLECER NUEVAMENTE COMO PREDETERMINADO Ó
2. TAMBIEN SE PUEDE ABRIR CON CLICK-DERECHO Opcion "Abrir con",
(elegir Google Chrome).
